Text a hashtag or username to a number, and receive an MMS of an Instagram photo from that hashtag or that user.






use Twilio and Asterisk





to run:
    * in one terminal window, run:
        ./zouni.rb
    * in another one, run:
        ./ngrok 4567
    * go to browser, localhost:4567, login, leave it alone
    * send text


1. username
    demo: @ruoyul, @lyyvalhalla for correct response
    demo: @ruoyil for a user that does not exist

2. tag
    demo: some random tags, I like #tbt, #foodporn

3. wrong format
    demo: $10 for something we don't support
